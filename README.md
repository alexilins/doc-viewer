# DocViewer

## Запуск

1. `nvm use`
2. `npm ci`
3. `npm run start`
4. перейти в `http://localhost:4200/document/1234`

## Доработки (из них исходят текущие ограничения)

- обработка случая когда в документе нет страниц
- добавление realtime многопользовательского редактирования через вебсокеты
- редирект на 404 страницу если документ на бэке не найден
- для скорости лучше использовать вместо jpeg страниц webp/avif
- перетаскивание аннотаций между разными страницами
- редактирование аннотаций
- инструменты редактирования и стилизации аннотаций (возможно WYSIWYG)
- добавление к модели документа его идентификаторв ID, аналогично для страниц и аннотаций - тогда для перемещения их между страницами не потребуется пересоздавать записи на бэке
- 2-страничный режим просмотра документов
- добавить экспорт (например в PDF) документа с аннотациями
- помимо аннотаций добавить возможность других интсрументов, например фигуры и free-form маркер
- адаптировать print mode для просмотрщика документов
- добавить кнопку сброса зума
- при удалении аннотации показывать таймер в течении которого можно откатить удаление, в целом можно добавить функционал undo/redo
